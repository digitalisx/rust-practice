# rust-practice
Practice for Rust programming language.

## Install
```shell
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

## Make project
```shell
cargo new project_name
```

## Compile and Run
```shell
cargo run
```
